# 📝Report
학업중에 작성했던 보고서입니다!<br/>
핵심내용은 MxNET 프레임워크를 이용한 선형회귀 객체지향 코드 분석을 주제로 작성되어있습니다.
<br />

## 1. 📊 MxNET을 이용한 선형회귀 모델 객체지향 구현

>> **기계학습(1) - PBL 프로젝트 | 이은우 (202025016)**  
>> MxNET을 이용한 선형 회귀 모델을 밑바닥부터 구현하고, 객체지향 코드 구조를 이해 및 분석
> 
> ## 🔍 주요 클래스 및 기능
> 
> ### 📌 `add_to_class` 데코레이터
> - 외부에서 정의된 함수를 클래스 내부에 추가할 수 있도록 지원
> 
> ### 📌 `HyperParameters` & `save_hyperparameters()`
> - 하이퍼파라미터 자동 저장
> 
> ### 📌 `ProgressBoard`
> - 학습 시각화(그래프 애니메이션) 도구
> 
> ### 📌 Module 클래스
> - 모든 모델의 기본 클래스 (학습/검증/시각화 기능 포함)
> - 주요 함수: __init__, loss, forward, plot, training_step, validation_step
> 
> ### 📌 DataModule 클래스
> - 데이터 다운로드, 배치 처리의 기반 클래스
> 
> ### 📌 SyntheticRegressionData 클래스
> - 인공 회귀 데이터 생성, 학습/검증 데이터 배치 제공
> ## 📎 부록
> - 결과 보고서: [`기계학습(1) 결과보고서.pdf`](https://github.com/DavidMao8849/Portfolio/blob/9a059dacebcffdba32b73c44aa85c569dcb53ef9/Report/0613%EA%B8%B0%EA%B3%84%ED%95%99%EC%8A%B5_PBL_%EA%B2%B0%EA%B3%BC%EB%B3%B4%EA%B3%A0%EC%84%9C(%EC%9D%B4%EC%9D%80%EC%9A%B0).pdf)
<br/>

>> **기계학습(2) PBL 프로젝트 | 이은우 (202025016)**  
>> 선형 회귀, 분류, CNN, 객체 인식 등 다양한 문제를 통해 MxNet 프레임워크의 핵심 기능을 학습
>
> ### ✅ 1. 선형 회귀 모델 학습 및 파라미터 확인
> - **설정**: w = [3, 1, 2], b = 4.2
> - **결과**: 학습 완료 후 추정된 w, b 값과 초기값 비교  
>   → w, b 모두 오차 0.1~0.2 수준, 학습 정상 작동
> ---
> ### ✅ 2. 프레임워크 API 구조 분석
> 
> #### 🧩 Model 클래스 주요 메서드
> - `__init__()`: 하이퍼파라미터 저장 및 초기화
> - `forward(X)`: 예측값 계산
> - `loss(y_hat, y)`: 손실 계산
> - `training_step(batch)`, `validation_step(batch)`: 학습/검증 수행
> 
> #### 🧩 Data 클래스
> - `get_dataloader(train)`, `train_dataloader()`, `val_dataloader()`
> #### 🧩 Trainer 클래스
> - `prepare_data()`, `prepare_model()`, `fit()`, `fit_epoch()`
> 
> ### ✅ 3. 분류 문제: One-hot Encoding, Softmax, Cross Entropy
> 
> - **입력**: [-0.4, 4.2, 0.3], 정답 라벨: 2
> - **Softmax** 결과: `[0.01, 0.97, 0.02]`
> - **Cross Entropy Loss**: ≈ 0.01
> 
> ### ✅ 4. CNN: Padding과 Stride 적용 결과
> 
> - 입력 이미지: 320x240, 커널: 5x5
> - **padding=0** → 출력: 316x236  
> - **padding=1** → 출력: 318x238
> 
> ### ✅ 5. 객체 인식: Bounding Box 시각화
> 
> - 예제 이미지에서 **사람과 개** 객체 인식
> - 눈대중으로 박스 좌표 추정 → bounding box 그리기
> - 잘못된 경우 박스 좌표 수정하여 재시각화
