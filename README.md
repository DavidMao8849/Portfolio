# 📝Projects
학업중에 진행했던 프로젝트들을 소개합니다! 
대부분의 프로젝트에서 팀장을 맡았었으며, 개인적으로 정리한 보고서도 존재합니다.
팀장의 역할로썬 프로젝트의 구조/계획 설계와 일정관리, 팀원들간의 의사소통, 문제해결 자문을 경험하였고,   
개발자 역할에서는 ***프로젝트 MVP와 관련된 핵심 기능과 알고리즘 개발***을 맡아 진행했습니다.
